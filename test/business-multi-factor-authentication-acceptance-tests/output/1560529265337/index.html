HM Revenue & Customs

Reset extra security

You can't sign in until your security settings have been reset.

You'll need to contact an admin user and ask them to remove extra security from your account.


How to remove extra security

An admin should log in to HMRC and select:

  1. Manage account
  2. Manage users
  3. Manage security against your name
  4. Remove extra security

You can reset extra security the next time you sign in.
